# Open World Game

Three.jsで作ったサイバー調の疑似オープンワールド試作です。ロボット型プレイヤー、順番に出現する3タイプの敵、HP、遮蔽壁、三人称カメラ、スマホ操作UIを含みます。

## 起動方法

`config.json` とJavaScriptモジュールを読み込むため、ローカルサーバー経由で開いてください。

```sh
python3 -m http.server 8000
```

ブラウザで以下を開きます。

```txt
http://localhost:8000/
```

## 役割

- `index.html`: HUD、操作説明、スマホ操作ボタン、横画面案内、CSS/JS読み込み
- `style.css`: HUD、ボタン、スマホ表示、背景などの見た目
- `main.js`: Three.jsの初期化、設定読み込み、ライト、メインループ、HUD更新、ポーズ、スコア、リザルト
- `enemy.js`: 近距離/中距離/遠距離の敵ロボット、遠距離スポーン、攻撃、撃破後リスポーン
- `player.js`: ロボットプレイヤー生成、移動、ジャンプ、接地、障壁衝突、走行アニメーション
- `skills.js`: 銃撃、設置型シールド、上昇、地雷、爆発エフェクト
- `world.js`: 地形チャンク、遮蔽壁、サイバー背景、疑似ランダム生成
- `input.js`: キーボード、マウス視点、スマホ用バーチャルスティック、スマホボタン、タッチ視点、横画面ロック
- `camera.js`: 三人称カメラ追従
- `utils.js`: 色変換、clamp、チャンク座標、乱数
- `config.json`: 速度、重力、チャンク、カメラ、入力感度などの調整値

## 操作方法

- `WASD` / 矢印: 移動
- Mouse: 視点操作
- `Space`: ジャンプ
- `Shift`: ダッシュ
- `1` / `J` / 左クリック: 銃撃
- `2` / `Q`: 数秒で消える設置型シールド
- `3` / `E`: 数秒間上昇
- `4` / `F`: 地雷を前方へ投げて設置
- `P` / HUDの`PAUSE`: 一時停止
- `R`: 初期位置へ戻る
- スマホ: 左側の円形バーチャルスティックで移動、画面ボタンでジャンプ、ダッシュ、各スキル、ポーズ、リセット

## 戦闘ルール

- プレイヤーと敵にはHPがあります。
- HPはHUDのバーと数値で表示され、ダメージに応じて減ります。
- 敵はプレイヤーから離れた場所に1体ずつ出現します。
- `Close`: 近距離型。接近して近接攻撃します。
- `Mid`: 中距離型。距離を詰めながら軽めの弾を撃ちます。
- `Far`: 遠距離型。距離を取りながら威力の高い弾を撃ちます。
- 敵HPを削り切ると、少し後に次のパターンの敵が遠くに出現します。
- プレイヤーHPが0になると、初期位置にリスポーンして残機が1減ります。
- 残機が3回分なくなるとゲームオーバーになり、倒した敵の数が最終スコアとして表示されます。
- 最高スコアはブラウザの`localStorage`に保存され、タイトル画面に表示されます。
- シールドは全身を覆える大きさで、敵弾を止めます。自分の弾は通過します。
- フィールド上のオブジェクトは遮蔽用の壁だけで、敵の射線を切るために使えます。

## 注意点

- `fetch("./config.json")` を使うため、直接ファイルを開くよりローカルサーバー経由を推奨します。
- Three.jsはCDNから読み込んでいるため、初回表示にはネットワーク接続が必要です。
- 地雷は弧を描いて前方へ投げられ、着地後に相手が触れるか、一定時間が経過すると爆発します。
